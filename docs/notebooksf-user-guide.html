Getting Started

What is NotebookSF?

NotebookSF is a research and note-taking assistant — a Library of Knowledge (LoK) — built natively on the Salesforce Platform. It helps you organize knowledge in a hierarchical fashion, similar to how a real library organizes books. Instead of flat notes scattered across folders, NotebookSF structures your learning materials into a clean hierarchy that prevents clutter and makes information easy to find.

Think of it as your personal Salesforce-native study hub: save reference links, write rich-text notes, study with flashcards, and test yourself with practice exams — all in one place with AI-powered assistance.

The Library Analogy

NotebookSF mirrors the way a real library is organized. Here's how the hierarchy works:

📚 Catalogs
Top-level collections (e.g., "Technical Stuff")
📂 Topics
Subject areas (e.g., "Analytics", "Security")
📓 Notebooks
Study workspaces (e.g., "Einstein Analytics")
📑 Sections
Groups within a notebook (e.g., "Prompt Builder")

Accessing the Application

NotebookSF runs as a Lightning Console Application called Notebook Console. To access it:

  1. Open the App Launcher (the grid icon in the top-left of your Salesforce org).
  2. Search for "Notebook Console" and select it.
  3. The Application Home Page will load, displaying your Catalogs in the left sidebar and Topic cards in the main area.

Tip: NotebookSF uses a console-style layout. When you open a notebook, it appears as a new tab — allowing you to switch quickly between notebooks and the home page.

2

Application Home Page

The Home Page is your starting point. It consists of two main areas: the Catalog Sidebar on the left and the Topic Cards grid in the center.

Catalog Sidebar

The sidebar displays all your Catalogs as a vertical navigation list. Click any catalog name to load its associated topics in the main area.

Managing Catalogs

+ Create Catalog

Click the "Add Catalog" button at the bottom of the sidebar. Enter a name in the modal and click Create.

✎ Edit Catalog

Hover over a catalog name to reveal the edit icon. Click it to rename the catalog via the edit modal.

✕ Delete Catalog

Hover over a catalog name to reveal the delete icon. Available only if the catalog contains no topics.

Topic Cards

When you select a catalog, the main area displays Topic Cards arranged in a columnar layout. Each Topic Card shows the topic's title and a list of its Notebooks (displayed as clickable links).

Managing Topics

+ Create Topic

Click the "Add Topic" floating button in the bottom-right corner. Enter a name and select a parent catalog.

✎ Edit / Reparent Topic

Hover over a topic card title to reveal the edit icon. Rename it or move it to a different catalog.

✕ Delete Topic

Hover over a topic card title to reveal the delete icon. Available only if the topic contains no notebooks.

Managing Notebooks

Notebooks appear as clickable links inside each Topic Card. They are your primary study workspaces.

+ Create Notebook

Click "Add Notebook" at the bottom of any Topic Card. Enter a name and select its parent topic.

✎ Edit / Reparent Notebook

Hover over a notebook link to reveal the edit icon. Rename it or move it to a different topic.

✕ Delete Notebook

Hover over a notebook link to reveal the delete icon. Warning: All contents will be deleted.

Opening a Notebook: Click any notebook title link to open it in a new console tab. The notebook will default to the Resources interaction mode.

Interaction Modes Overview

Once inside a notebook, you have access to four specialized Interaction Modes — each designed for a different type of learning activity. Switch between them using the mode selector buttons near the top of the page.

Resources

Save and organize external URL links — web pages, PDFs, and videos — for quick reference.

Notes

Create rich-text notes with full formatting. Write summaries, key takeaways, and study materials.

Cards (Flashcards)

Two-sided flashcards for active recall. Flip them to study, link supporting resources, and sort them in any order.

Exams

Multiple-choice practice questions with explanations. Test your knowledge and review supporting resources.

Shared Layout Elements

All interaction modes share a consistent layout:

Section Cards

Items are organized into Section Cards displayed in columns. Each card has a title, items, and action buttons.

Bookmarked Section

A special section card at the top of every list view showing all bookmarked items for quick access.

Agent Sparky Panel

An AI chat assistant on the right side of the page, always available to help you search and find information.

4

Resources Mode

Resources Mode lets you collect and organize external references — web links, PDFs, and videos — organized by section. Each resource is a clickable item that opens its URL in a new browser tab.

List View

The Resources List View displays Section Cards arranged in columns. Each card shows the section title, a percentage indicator, and a list of resource items with their titles and link icons.

Each resource item has three action icons:

✎ Edit — appears on hover ☆ Bookmark — always visible 🗑 Delete — always visible

Working with Resources

+ Adding a Resource

Click the "Add Resource" button at the bottom of any Section Card. In the Create Resource modal, enter the Resource Link (URL) and Title. Select the resource type using the type buttons: Link, PDF, or Video. Click Create to save.

✎ Editing a Resource

Hover over any resource item to reveal the edit icon. Click it to open the Edit Resource modal where you can update the link and title.

✕ Deleting a Resource

Click the delete icon on any resource item. A confirmation modal will appear warning that all content associated with the resource (notes, cards, exam questions referencing it) will also be removed.

Managing Sections

Sections are the organizational groups within the Resources List View (and all other modes). You can manage them from the Resources view:

Add Section

Click the "Add Section" floating button in the bottom-right corner of the page.

Edit Section

Hover over a section card title to reveal the edit icon. Rename the section.

Delete Section

Hover over a section card title to reveal the delete icon. All contents within the section will be deleted.

5

Notes Mode

Notes Mode provides a rich-text note-taking experience. Create notes, organize them by section, and write detailed study materials with full formatting support.

List View

The Notes List View displays Section Cards with note items. Each note item shows the note's title. Click any note to open it in the Detail View.

Creating a Note

Click "Add Note" on any Section Card. In the Create Note modal, enter a title and select the section. The note will be created and you'll be taken directly to the Detail View to start writing.

Editing a Note Title

Hover over a note item to reveal the edit icon. Click it to rename the note via the Edit Note modal.

Deleting a Note

Click the delete icon on any note item. A confirmation modal will ask you to confirm deletion.

Detail View — Rich Text Editor

The Detail View is a full-page rich text editor for writing and editing note content. The toolbar at the top provides formatting options:

B I U ⇧ Indent ≡ Align # Heading ⊞ List T Format

Key features of the Detail View:

→ Note Title appears at the top. Hover over it to reveal an edit icon for renaming.
→ Save & Close button at the bottom-right saves your changes and returns to the List View.
→ Cancel button at the bottom-left discards unsaved changes and returns to the List View.

Important: Always use Save & Close before leaving a note to ensure your changes are saved. Navigating away without saving will discard your edits.

Cards (Flashcards) Mode

Flashcards are two-sided study tools for active recall. Each card has a front side (question, prompt, or term) and a back side (answer or definition). Cards can also reference supporting resources from your notebook.

List View — Deck Overview

The Cards List View displays Section Cards showing deck statistics: the section title, total number of cards in the deck, and last modified date. Each Section Card has an "Open Deck" button.

Einstein Trust Layer (15%)

Total cards in deck: 12

Last Modified: 04/25/2026

☆ ✎
Open Deck

Detail View — Study Carousel

The Detail View displays flashcards one at a time in a carousel. Here's how to interact with them:

Click or press Space to flip the card and reveal the other side
← → Use the arrow buttons at the bottom center to navigate between cards
Close Click the Close button (bottom-right) to return to the List View

Action buttons at the top-right of the Detail View:

+ Add ✎ Edit ⇅ Sort ✕ Delete

Creating & Editing Cards — 3-Page Wizard

Clicking Add or Edit launches a three-page wizard for building or modifying flashcards:

1

Front Side

Enter the question, prompt, or term for the front of the card. Text automatically scales down to fit without scrolling. Use Cancel to abort or Next to proceed.

2

Back Side

Enter the answer or definition using a rich text editor. You can format text with bold, italics, lists, and more. Use Back to revisit the front or Next to continue.

3

Supporting Resources

Select resources from the notebook to link to this card. Use the Resource Selector dropdown to add items, and the sort buttons to order them. Click Save to commit all data or Cancel to discard.

Tip: The Save button only appears on Page 3. You must complete all three pages to save a card. Clicking Cancel at any point discards all changes.

Additional Card Features

Sorting Cards

Click the Sort action button from the Detail View to open the Card Sort View. Drag cards into your preferred order using the sort buttons, then click Save to apply or Cancel to discard.

View Referenced Resources

At the bottom of the Detail View, click "View Resources" to see all resources linked to the current card. Each resource is clickable and opens in a new tab.

Exams Mode

Exams Mode lets you create and take multiple-choice practice exams. Questions are presented one at a time with answer choices. After submitting, you see the correct answer, explanation, and supporting resources.

List View — Exam Overview

Similar to Cards, the Exam List View shows Section Cards with exam statistics: total questions and last modified date. Each card has an "Open Exam" button to launch the exam.

Detail View — Taking an Exam

Questions are displayed one at a time. Here's the exam-taking workflow:

1

Read the question displayed at the top of the page.

2

Select an answer from the multiple-choice options (3 to 6 choices labeled A, B, C, etc.).

3

Click "Submit" to check your answer.

4

Review the result — the correct answer is highlighted in green, along with a detailed explanation and list of related resources.

Navigation: ← → Arrow buttons to move between questions Close button returns to List View

Creating & Editing Questions — 2-Page Wizard

Click Add or Edit from the Detail View to launch the question wizard:

1

Question & Answers

Enter the question text. Define between 3 and 6 answer candidates. Select which answer is correct. Write an explanation that will be shown after the user submits. Click Next to proceed or Cancel to abort.

2

Supporting Resources

Select resources from the notebook to provide context for the question. Sort them to define the display order. Click Save to commit or Back to return to Page 1.

Additional Exam Features

Sorting Questions

Click the Sort action button to open the Question Sort View. Rearrange questions using the sort buttons, then Save or Cancel.

Bookmarking Questions

Use the bookmark toggle on any question in the Detail View to add it to the Bookmarked Exams section for quick revisiting.

8

Bookmarks

Bookmarks help you avoid "category clutter" by providing quick access to your most important items. The feature works consistently across all four interaction modes.

How Bookmarking Works

Each item (resource, note, card, or exam question) has a bookmark toggle icon (star icon) visible on the item row.
Click the star to toggle bookmarking. A filled star means the item is bookmarked.
Bookmarked items appear in a dedicated "Bookmarked" Section Card at the top of every List View for instant access.

Tip: Use bookmarks to collect your most challenging flashcards or frequently referenced resources in one place — no matter which section they belong to.

Agent Sparky (AI Chat)

Agent Sparky

Agent Sparky is your AI-powered assistant, always visible in a panel on the right side of the notebook view. It leverages Retrieval Augmented Generation (RAG) to search and find information across all your catalogs, topics, notebooks, and sections.

What Agent Sparky Can Do

→ Search across your entire knowledge base for specific information
→ Provide summarized answers from your resources and notes
→ Help generate study materials like flashcards and exam questions
→ Answer questions about your Salesforce data

Suggested Prompts

Agent Sparky provides quick-start suggested prompts you can click to get started:

Show me my recent opportunities What are my top accounts? Create a new lead Analyze my sales pipeline

Pro Tips

📚 Organize First

Set up your Catalog → Topic → Notebook hierarchy before adding content. A well-organized structure makes it much easier to find information later.

🔗 Resources Are Foundation

Start by adding resources (URLs) to your notebook. Cards and exam questions can reference these resources, creating a rich learning web.

⭐ Bookmark Strategically

Bookmark items you need quick access to — difficult flashcards, key resources, or exam questions you got wrong. They appear at the top of every list.

🤖 Ask Agent Sparky

Use Agent Sparky to quickly search across your entire library. It can find information from any catalog, topic, or notebook — saving you time browsing.

🔀 Hover to Discover

Many edit and delete buttons are hidden by default and only appear when you hover over an item. This keeps the interface clean while still providing quick access.

📋 Use Console Tabs

NotebookSF uses the Salesforce Console layout. Open multiple notebooks as tabs and quickly switch between them without losing your place.
